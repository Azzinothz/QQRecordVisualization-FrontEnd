# QQ聊天记录可视化

## 介绍
QQRecordVisualization可以接收用户的QQ聊天记录文件，将其解析为结构化数据存入数据库，并输出JSON类型的Option配置供前端Echarts使用。

目前支持的可视化类型有：
- 词云（Word Cloud）
